# 雷军完整时间线（1969-2026）

## 第一部分：早年与金山时期

| 时间 | 事件 | 对雷军思维的影响 |
|------|------|-----------------|
| 1969年12月16日 | 出生于湖北省仙桃市，父亲是教师 | 从小受"知识改变命运"教育，形成勤奋刻苦的性格 |
| 1987年 | 考入武汉大学计算机系 | 接触信息技术前沿 |
| 1987年 | 在武大图书馆读到《硅谷之火》，被乔布斯的故事深深触动 | **思想转折点①**——确立"创办伟大公司"的人生梦想 |
| 1987-1989年 | 两年修完全部学分；大二写的加密软件BITLOK被金山采用 | 确立"技术天才"的自我认同 |
| 1991年 | 毕业，被分配到北京近郊一家研究所 | 感受体制内束缚，决心投身商业 |
| 1991年底 | 在北京计算机展结识求伯君，受邀加入金山 | **思想转折点②**——选择"先跟对人"的务实策略 |
| 1992年 | 正式加入金山软件，成为第6号员工 | 开始16年的金山生涯 |
| 1998年 | 28岁出任金山总经理，成为IT行业最年轻的总经理之一 | 从纯技术转向管理 |
| 1999-2003年 | 主导WPS对抗微软Office的持久战 | 形成"在巨头阴影下求生存"的战斗意识 |
| 2007年10月9日 | 金山在香港上市（3888.HK） | 完成使命，但内心感到迷茫 |
| 2007年12月 | 辞去金山CEO | **思想转折点③**——"金山上市了，但我并不开心"。从"执行者"转向"思考者" |

## 第二部分：天使投资时期（2007-2010）

| 时间 | 事件 | 对雷军思维的影响 |
|------|------|-----------------|
| 2007-2010年 | 投资约20家企业：UCWeb、YY、拉卡拉、凡客诚品、乐淘等 | 大量接触不同赛道创业者，形成系统化的互联网商业理解 |
| 2008年 | 投资UCWeb并出任董事长 | 深入理解移动互联网趋势 |
| 2009-2010年 | 提炼出"顺势而为"的投资哲学 | **思想转折点④**——"站在风口上，猪都能飞起来"。从"逆风作战"转向"借势发展" |

## 第三部分：小米创立与早期发展（2010-2015）

| 时间 | 事件 | 对雷军思维的影响 |
|------|------|-----------------|
| 2010年4月6日 | 与林斌等7位联合创始人创立小米公司 | **思想转折点⑤**——将梦想付诸实践 |
| 2010年8月 | 发布MIUI第一版，面向100个发烧友用户 | 确立"用户参与开发"的互联网思维 |
| 2011年8月16日 | 小米手机1发布，定价1999元 | 开创"极致性价比"商业模式 |
| 2012年 | 全年销售719万台，营收126亿元 | "风口论"获实践验证 |
| 2014年 | 出货6112万台，估值450亿美元 | 模式巅峰，但开始反思过度追求规模的问题 |
| 2015-2016年 | 遭遇增长瓶颈，被华为反超 | **思想转折点⑥**——"至暗时刻"，亲自接管手机研发和供应链 |

## 第四部分：小米二次创业（2016-2020）

| 时间 | 事件 | 对雷军思维的影响 |
|------|------|-----------------|
| 2016年10月 | 发布小米MIX全面屏手机，被芬兰设计博物馆收藏 | **思想转折点⑦**——从"追随者"变为"创新者" |
| 2018年7月9日 | 小米在香港上市（1810.HK），上市即破发 | 思考长期价值与短期市场评价的张力 |
| 2019年7月 | 进入世界500强，成为最年轻的世界500强（成立9年） | 证明互联网模式可打造世界级企业 |
| 2020年2月 | 小米10系列发布，定价3999-5999元，正式冲击高端 | **思想转折点⑧**——打破"只能卖1999"的自我设限 |

## 第五部分：小米造车（2021-2024）

| 时间 | 事件 | 对雷军思维的影响 |
|------|------|-----------------|
| 2021年3月30日 | 宣布小米造车，首期投资100亿元 | **思想转折点⑨**——"人生最后一次重大创业"，赌上全部声誉 |
| 2021年 | 收购DeepMotion（深动科技），补强自动驾驶 | 认识到智能驾驶是造车核心 |
| 2023年12月28日 | 小米汽车技术发布会 | 正式向市场展示造车成果 |
| 2024年3月28日 | SU7正式上市，定价21.59万元 | **思想转折点⑩**——小米成为横跨消费电子和汽车的科技公司。雷军亲自为车主开车门 |
| 2024年4月 | SU7首月锁单超8万台 | 验证小米品牌跨品类能力 |
| 2024年10月 | SU7 Ultra原型车纽北圈速6分46秒 | 证明小米技术实力 |
| 2024年全年 | SU7交付约13万台；集团营收超3600亿元 | "手机+汽车+IoT"三驾马车形成 |
